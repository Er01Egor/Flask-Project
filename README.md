# Smart Kitchen

## Для кого проект

- **Тем, кто не знает, что приготовить**
- **Новичкам и студентам**
- **Людям на диете или со специальным питанием**
- **«Фуд-эстетам» и любителям вдохновения**
- **Экономным хозяевам**

## Основные функции

- **Умный поиск:** Подбор блюд по списку имеющихся ингредиентов.
- **База рецептов:** Пошаговое описание приготовления с фотографиями.
- **Личный кабинет:** Возможность сохранять любимые рецепты в "Избранное".
- **Авторские блюда:** Возможность добавлять свои авторские блюда.

## Перед запуском _локально_

*Следуйте следующей инструкции, чтобы правильно запустить проект:*
> **В Командной строке:**

- **Скачиваем проект:**

```
curl -L https://github.com/Er01Egor/Flask-Project/archive/main.zip -o project.zip
```

- **Переходим в папку проекта:**

```
cd Flask-Project
```

- **Устанавливаем зависимости:**

```
pip install -r requirements.txt
```

- **Создаем БД:**

```
python -m data.add_db
```

- **Запускаем проект:**

```
python main.py
```

> **В любом браузере вводим:**

```
http://127.0.0.1:8080
```

## For whom the project is intended

- **Those who do not know what to cook**
- **Beginners and students**
- **People on a diet or with special nutrition**
- **"Food-aesthetes" and lovers of inspiration**
- **Economical hosts**

## Main functions

- **Smart search:** Select dishes based on the list of available ingredients.
- **Recipe database:** Step-by-step cooking instructions with photos.
- **Personal account:** The ability to save your favorite recipes in "Favorites".
- **Author's dishes:** The ability to add your own author's dishes.
